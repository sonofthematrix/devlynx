Privacy Policy

Last updated: March 2026

Short version: We don't sell your data, track your browsing, or store your API key. The extension runs in your browser. License verification is the only data that touches our server.

Who we are

DevLynx AI is a Chrome browser extension that provides AI-powered developer tools. This policy explains what data is collected when you use the extension and our website.

What we do NOT collect

  • Your browsing history or the URLs you visit
  • The content of pages you inspect or modify
  • Your OpenAI API key — it is stored only in your browser's local storage and is never transmitted to our servers
  • Any personal data for advertising or profiling purposes
  • Keystroke data or form inputs

Your OpenAI API key (BYOK)

If you choose to use your own OpenAI API key (Bring Your Own Key), that key is stored exclusively in your browser using the Chrome Storage API. It is sent directly from your browser to OpenAI's servers when you use AI features. It never passes through the DevLynx server.

What our server does receive

The DevLynx feed server (hosted on Vercel) receives the following when you use server-side features:

  • License verification: your Gumroad license key and a device ID generated locally in your browser, used to enforce the device limit on your license
  • Trial tracking: an anonymous device ID used to count your free AI trial uses — no name, email, or account required
  • AI requests: the text prompt you send (e.g. a question or error message) when using server-side AI. This is forwarded to OpenAI and not stored by us
  • Screenshots: if you use the screenshot feature with server storage enabled, the image is stored in Vercel Blob storage tied to your session
  • Health checks: the extension pings /health to verify the server is reachable — no user data is sent

Device ID

A random device identifier is generated in your browser the first time you use DevLynx AI. This ID is used only to track your trial usage count and enforce license device limits. It is not linked to your identity, email address, or any other personal information.

License and payments

Payments are processed by Gumroad. When you purchase DevLynx Pro, Gumroad handles your payment information according to their own privacy policy. We receive only your license key and optionally your email address from Gumroad for support purposes.

Third-party services

  • OpenAI — receives your AI prompts when you use AI features. See OpenAI's privacy policy.
  • Vercel — hosts the DevLynx feed server. See Vercel's privacy policy.
  • Gumroad — handles payments and license key delivery. See Gumroad's privacy policy.

Data retention

  • Trial usage counts are stored in Vercel Blob and expire after 90 days of inactivity
  • License activation records are kept for the duration of your license to enforce device limits
  • Screenshots stored via server storage can be deleted by clearing your browser data or contacting support

Chrome Web Store permissions

DevLynx AI requests the following permissions and uses them solely for the stated purpose:

  • storage — save your API key, license, trial count, and site mods locally in your browser
  • scripting — inject CSS/JS mods you create with the AI Mod Generator into the current page
  • activeTab / tabs — read the current page's URL and content to provide contextual AI answers
  • contextMenus — add right-click menu options for quick AI actions
  • downloads — allow you to download screenshots
  • host_permissions (<all_urls>) — required so the API tester can send requests to any endpoint you specify, and so AI tools work on any website

Your rights

You can delete all locally stored data at any time by going to Chrome Settings → Extensions → DevLynx AI → Clear data, or by uninstalling the extension. For server-side data (trial count, license activation), contact us and we will delete it within 30 days.

Changes to this policy

If we make material changes to this policy, we will update the date at the top of this page. Continued use of the extension after changes constitutes acceptance of the updated policy.

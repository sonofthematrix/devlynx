# Monetization-ideeën (hypotheses — niet store copy)

Alles hieronder is **richting** voor later onderzoek. Geen vaste prijzen in de Chrome Web Store of op de landing page tenzij je het echt verkoopt en support kunt leveren.

---

## Mogelijke Pro-tier uitbreidingen (concept)

| Idee | Opmerking |
|------|-----------|
| Team dashboard | Vereist accounts, backend, support-last |
| Historical tracking | Opslag + privacy + GDPR-overwegingen |
| API access tot eigen backend | Alleen als je een stabiele API + keys / abuse prevention hebt |
| White-label reports | Enterprise sales-cyclus; juridiek review |

## Enterprise (ver vooruit)

- SAML / SSO
- Audit logs
- Custom model training — zelden haalbaar voor kleine teams; meestal “custom prompts” eerst

---

## Praktisch advies

1. **Launch** met huidige model (Free trial + lifetime Pro) zoals in de app.  
2. **Meet** wat gebruikers willen betalen **na** echte usage (support tickets, Gumroad feedback).  
3. Voeg **subscripties** pas toe als terugkerende kosten (hosting, support) dat rechtvaardigen.
